# nextjs-todolist

This is a simple to-do list app built with Nextjs and Prisma.
Mainly using [server action](https://nextjs.org/docs/app/building-your-application/data-fetching/server-actions) to handle DB operations.
## Tech Stack

- Next.js
- Prisma
- PostgreSQL
- Tailwind CSS
- Framer Motion
